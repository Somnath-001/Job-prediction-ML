💼 AI Career Recommendation System

A machine learning-based career recommendation system that predicts suitable job roles based on a user’s education level, age, and skills.

Features

* Job role prediction using Random Forest
* Skill-based career recommendations
* Career readiness score
* Skill gap analysis
* Training recommendations
* Top 3 career matches
* Interactive Streamlit interface

Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* Joblib
